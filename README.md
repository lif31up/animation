# 에니메이션 실험소
웹 프로그래밍에서 에니메이션을 사용하기 위한 요소를 개발하거나 그와 관련된 공부 및 연구를 하기 위해 존재하는 웹사이트입니다.
## 웹사이트를 방문하기
오른쪽의 리포지토리 설명에 기술되어있는 링크를 클릭해 해당 `next.js` 응용의 빌드를 확인합니다.
## 요소를 확인하기
웹사이트에 전시된 요소와 관련된 논리는 전부 `component`에 위치합니다.
